# JXCIX TOOLS

Windows cybersecurity and system administration toolkit written in Python.

## Features

* Ping host
* DNS lookup
* SSL certificate checker
* Network connections
* File hash checker
* Duplicate finder
* Password generator
* Username generator
* UUID generator
* Base64 encoder / decoder
* PC information
* System dashboard
* Windows audit
* Log management

## Installation

```bash
pip install rich psutil
```

## Usage

```bash
python jxcix_tools_top.py
```
